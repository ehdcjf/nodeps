# 1010 다리 놓기

dp[i][j] 동쪽 i 개 사이트에서 서쪽 j개 사이트에 지을 수 있는 다리의 경우의 수

# 9625 BABBA

dp[i][a] 는 버튼을 i번 눌렀을 때, A의 개수
dp[i][b] 는 버튼을 i번 눌렀을 때, B의 개수

조건식
dp[i][a] = dp[i-1][b] (i>0);
dp[i][b] = dp[i-1][a] + dp[i-1][b] (i>0);

초기값
dp[0]=[1,0]
